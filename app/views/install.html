Minimum

  • Internet connectivity
  • Ubuntu 16.04
  • Docker
  • 1 GiB Storage
  • 1 GiB RAM

Recommendation

Hostero requires Docker to work. We recommend to install Docker via our script, but you can install it using alternative ways as well.

Install the Universal CPU Miner

Run this command in the Terminal:

curl https://www.hostero.eu/install?a={{account.name}} | sh

Optional

Cloud providers like AWS, Google, etc, have an option to set a start-up script. The script will execute after the VPS is created, so you don't have to manually install it.

Add this script to user data:

#!/bin/bash
curl https://www.hostero.eu/install?a={{account.name}} | sh

Note: The commands are specific to your account, do not share them with others

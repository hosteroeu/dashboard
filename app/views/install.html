Requirements for Host:

  • Internet connectivity
  • Linux (we recommend Ubuntu 16.04)
  • Docker
  • 1 GiB RAM
  • 1 GiB Storage

Install Docker

Before installing Hostero, you first need to install Docker. We recommand to install Docker using our instructions, but you can install it using alternative ways as well.

Run this command to install it:

curl https://releases.rancher.com/install-docker/17.12.sh | sh

Note: This command is specific to your account, do not share it with others

Note: You need to have curl installed. If it's not installed, run: apt install curl

Install Hostero

The client is essentially a Docker container that runs on your Host.

Run this command to install it:

sudo docker run -e CATTLE_HOST_LABELS='account={{account.name}}' --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU

Note: After you install the Hostero client, wait a few minutes for the host to appear in the Hosts page.

Alternative

User data script

#!/bin/bash
curl https://releases.rancher.com/install-docker/17.12.sh | sh

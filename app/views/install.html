Installation in two easy steps (view video):

  1. Install Docker

    curl https://releases.rancher.com/install-docker/17.12.sh | sh

  2. Install Hostero

    docker run -e CATTLE_HOST_LABELS='account={{account.name}}' --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/8A2F48145D7D544876F2:1514678400000:FxaguPYvHQzDvi30O5NBu8FBpMo

Requirements:

  • Ubuntu 16.04
  • 0.1 GiB RAM / host
  • 1 GiB SSD/HDD / host
  • 1 miner / host
